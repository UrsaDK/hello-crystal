# Hello World: Crystal

- [Optional] Install all dependencies:
  ```
  make lib
  ```

- Run all the tests and create a development binary:
  ```
  make
  ```

  The above command is the equivalent of running

  ```
  make tests
  make targets
  ```

- Create a new release binary:
  ```
  make release
  ```

- Remove all build artefacts:
  ```
  make clean
  ```
